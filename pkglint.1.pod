=head1 NAME

pkglint - detect common mistakes and stylistic issues in pkgsrc
package definitions

=head1 SYNOPSIS

B<pkglint> [I<OPTION>]... [I<PKGSRCDIR>]...

=head1 DESCRIPTION

B<pkglint> runs a series of checks on one or more pkgsrc directories
to detect common mistakes and stylistic issues in C<pkgsrc> package
definitions. B<pkglint> follows the requirements of L<Pkgfile(5)>, but
also contains additional checks.

=head1 OPTIONS

=over 8

=item B<-d>, B<--dead-links>

Check F<Pkgfile> and F<README> files for dead links.

=item B<-i>, B<--invalid>

Check F<.footprint> file for invalid directories, like F</usr/local>,
F</usr/share/locale>, F</usr/share/doc>, F</usr/share/info>, etc.

=item B<-j>, B<--junk>

Check F<.footprint> file for junk files, like F<README>, F<TODO>,
F<NEWS>, etc.

=item B<-s>, B<--suid-sgid>

Check F<.footprint> file for SUID/SGID files and directories.

=item B<-w>, B<--world-writeable>

Check F<.footprint> file for world-writeable files and directories.

=item B<-v>, B<--version>

Print version and exit.

=item B<-h>, B<--help>

Print help and exit.

=back

=head1 EXIT STATUS

0 if success, >0 otherwise.

=head1 SEE ALSO

L<finddeps-linked(1)>, L<findredundantdeps(1)>

=cut
