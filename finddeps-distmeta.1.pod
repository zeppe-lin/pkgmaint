=head1 NAME

finddeps-distmeta - parse Perl-module's distmeta file and find
dependencies

=head1 SYNOPSIS

B<finddeps-distmeta> [I<OPTION>] [FILE]

=head1 DESCRIPTION

B<finddeps-distmeta> parses so-called "distmeta" (CPAN distribution
metadata) of Perl-module and find dependencies.  It shows dependencies
that are in the core Perl distribution and searches missing modules
in the existing installed p5-* packages.

Currently, B<finddeps-distmeta> parses only F<META.yml> file, which
is generated by C<ExtUtils::MakeMaker>.  The parsing of F<META.json>
will be added soon.

B<finddeps-distmeta> operates on packages' sources repository to
search dependencies in the packages' footprints, and therefore
requires L<pkgman(1)>.

=head1 OPTIONS

Mandatory arguments to long options are mandatory for short options
too.

=over 15

=item B<-v>, B<--version>

Print version and exit.

=item B<-h>, B<--help>

Print help and exit.

=back

=head1 EXAMPLES

The typical usage of B<finddeps-distmeta> is to call it from the
Perl's module directory:

  cd /var/cache/pkgmk/sources/URI-5.17
  finddeps-distmeta

Or to specify the location of F<META.yml> file manually:

  finddeps-distmeta /var/cache/pkgmk/source/URI-5.17/META.yml

Another convenient way is to call B<finddeps-distmeta> from the
F<Pkgfile> when you're building the package:

  build() {
          cd URI-$version
          finddeps-distmeta; exit 1
          ...
  }

So, you can check the output, fix "Depends on:" field and remove
unnecessary line.
